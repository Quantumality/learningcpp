1. Introduction

Ce cours a pour but de vous faire apprendre la programmation avec C++, l'un des langages les plus utilisé dans le monde.

A qui s'adresse ce cours ?

A tout le monde ! N'importe qui peut apprendre la programmation avec C++, c'est même le meilleur langage pour commencer la programmation, alors oui vous avez aussi Python qui lui est plus facile, mais sachez que si vous connaissez C++ vous aurez plus facile à apprendre d'autres langages de programmation dans le futur.
